Two things to be done:
    - set myvundle path
    - add this folder to runtime and rune myvimrc script

## linux 

    let MYVUNDLE='/c/vim-settings/bundle'
    set runtimepath+=/c/vim-settings
    runtime myvimrc


## windows

    let MYVUNDLE='c:\vim-settings\bundle'
    set runtimepath+=c:\vim-settings
    runtime myvimrc
